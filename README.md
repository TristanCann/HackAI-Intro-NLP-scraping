# HackAI: Introduction to Python, text as data and collecting it from the web

This repository contains a collection of materials to get people started with Python, natural language processing and collecting data from the Internet.

Each of the modules listed below is designed to work on Google Colab, and in the following suggested order. Feel free to skip ahead if you're already comfortable with one of the sections.

## Section 1: Introduction to Python
The two files here will introduce the basics of Python programming and allow you to check everything is running correctly in Google Colab. There are a set of exercises (with suggested solutions) scattered throughout the two files. `Intro_to_Python.ipynb` will introduce variables, types, functions and loops in Python. `Intro_to_Python_exercises.ipynb` will set you a series of challenges to apply the skills you have learnt and test your understanding.

## Section 2: Scraping data from the web
An example of how to collect data from the web is included in `Scrape_webpages_parliamentary_debates.ipynb`. This will take you through the process of collecting speechs in the UK Parliament and separating them based on the party of the speaker.

## Section 3: Natural language processing
The selection of files labelled `NLP` will introduce several common ways to work with text data. In `NLP-Text cleaning and searching.ipynb` you'll learn why it is important to clean your text data and how to do it. In `NLP-Vectorisation, embeddings and similarity.ipynb` you'll learn how to convert text to quantitative data and measure the semantic content across large corpora of texts. In `NLP-Text classification.ipynb` you'll learn how to use pretrained models for a range of text annotation tasks, and how to work with the extensive library of models on HuggingFace. In `NLP-Topic modelling.ipynb` you will learn how to apply topic modelling using text embeddings - a useful tool for summarising and interpreting large text datasets. Solutions to the exercises within are also provided.
